# Projeto Biblioteca Digital

Esse é um projeto feito por estudantes do segundo semestre de Ciência da Computação para disciplina de Programação Orientada a Objetos.

# Estrutura atual do projeto

```
.
└── src
    ├── enums
    │   ├── StatusEmprestimo.java
    │   └── StatusExemplar.java
    ├── main
    │   └── Main.java
    ├── model
    │   ├── Autor.java
    │   ├── Emprestimo.java
    │   ├── Exemplar.java
    │   ├── Livro.java
    │   ├── Pessoa.java
    │   └── Usuario.java
    ├── repository
    │   ├── AutorRepository.java
    │   ├── EmprestimoRepository.java
    │   ├── ExemplarRepository.java
    │   ├── LivroRepository.java
    │   └── UsuarioRepository.java
    └── services
        └── EmprestimoService.java
```
